# tetris-python
Classic Tetris game built in Python using Pygame. Without using LLMs.

## Features
- Piece rotation
- Collision detection
- Line clearing

## How to run
1. Install dependencies:
   pip install pygame

2. Run the game:
   python tetris.py

## Known Issues
- Rotation system does not fully handle wall kicks, which can cause pieces to overlap boundaries.
- Collision detection may fail in rare edge cases when pieces stack rapidly.
